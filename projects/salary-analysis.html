Salary Data Analysis
& Exploratory Data Analysis

Personal Data Analysis Project

This project performs a comprehensive Exploratory Data Analysis (EDA) on a real-world salary dataset containing 22,770 records from multiple companies, locations, job roles, and employment types. The goal is to uncover salary trends, understand compensation factors, and answer real business intelligence questions through data-driven analysis and visualization — including a 4-panel analytics mini dashboard.

Real-WorldData Source

Key columns: job_title, job_roles, salary, location, company_name, rating, employment_status, salaries_reported

Data Cleaning

  • Loaded and inspected dataset: shape, head, tail, dtypes, describe()
  • Identified missing values in company_name column
  • Checked and confirmed no duplicate rows
  • Standardized column names (lowercase + underscores)
  • Outlier removal using IQR Method: calculated Q1, Q3, IQR → removed values below Q1 - 1.5×IQR and above Q3 + 1.5×IQR
  • Validated outlier removal with before/after boxplot comparison

Key Findings

💼 Data roles pay well

Data Science and Engineering-related job roles consistently rank among the highest-paying categories in the dataset.

🏙️ Location matters significantly

Metro cities like New Delhi offer substantially higher average salaries compared to smaller cities.

⭐ Rating weakly correlates with salary

There is a slight positive correlation between company rating and salary — but it is weak, meaning high-rated companies don't always pay the most.

📋 Employment type has a big impact

Full-time employees earn considerably more on average than part-time or contract workers in the same roles.

📊 After IQR outlier removal

The salary distribution becomes more normally shaped, making statistical analysis more reliable and representative.
