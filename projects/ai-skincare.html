Overview

BeauTech is an AI-powered skincare analysis platform developed for the Huawei ICT Competition. Users can upload a photo or use their webcam to have their facial skin analyzed in real time. The system identifies skin conditions and recommends personalized skincare products matched to the user's skin type and concerns.

The platform combines computer vision, deep learning, and a recommendation engine to deliver a seamless skincare journey — from analysis to product discovery — all in one web application.

Key Features

Face Skin Analysis

Real-time skin condition detection via photo upload or live webcam

AI Recommendations

Personalized skincare products matched to your detected skin type

Progress Dashboard

Track your skin health progress over time with visual analytics

Responsive Web App

Fully responsive design accessible on desktop and mobile devices

Business Model

🆓 Freemium

  • Basic skin condition analysis
  • General product recommendations
  • Free to all users

💎 Premium

  • Advanced skin analysis reports
  • Personalized skincare routines
  • Progress tracking dashboard

Technical Implementation

  • Computer vision model trained to detect skin conditions (acne, dryness, oiliness, sensitivity)
  • Deep learning pipeline for facial feature extraction and skin type classification
  • Recommendation engine that maps detected conditions to curated product database
  • Web frontend with webcam integration for real-time analysis
  • Backend API connecting the ML model to the web interface
  • Product database with match scoring (e.g., 67% match for CeraVe Hydrating Cleanser)
